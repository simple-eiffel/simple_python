Overview

simple_python is a production-ready Eiffel library for communicating with Python orchestration systems. It provides three pluggable transport protocols: HTTP, IPC (Windows named pipes), and gRPC.

Perfect for validation workflows where Eiffel hardware control boards need to communicate with Python machine-learning pipelines with guaranteed ultra-low latency (≤10ms p95 for 1KB payloads via IPC).

Part of the Simple Eiffel ecosystem.

Features

  • Multiple Transports: HTTP, IPC (Windows pipes), gRPC (pluggable design)
  • Message Types: Validation requests/responses with custom attributes
  • Serialization: JSON and binary formats with 4-byte length prefix framing
  • SCOOP-Safe: Concurrent-ready message handling for multi-threaded Eiffel systems
  • Design by Contract: Complete preconditions, postconditions, and invariants
  • Void-Safe: Full null-safety with detachable/attached type checking
